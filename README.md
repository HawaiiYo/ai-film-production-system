## 📖 项目简介

**AI电影工坊**是一个基于Claude Code的**多Agent AI电影制作协作系统**。它模拟真实电影工业的完整流程——从导演策划、剧本创作、角色/场景设计、分镜故事板到最终AI视频生成——通过多个专业化AI Agent协作，将你的故事创意转化为结构化、可执行的AI视频制作方案。

### 为什么需要这个系统？

市面上的AI视频工具（Runway、Pika、Kling、Sora、Seedance等）都是**单点工具**。你用A工具生成角色图，用B工具生成场景，用C工具生成视频，但没有人帮你确保：
- 角色在不同镜头中长得一样
- 场景风格和导演风格保持一致
- 分镜脚本是真正可执行的

**AI电影工坊**填补了这个空白：它用Multi-Agent协作来模拟一个真正的"虚拟剧组"，从创意到执行全程保障一致性和高质量输出。

### ✨ 核心特性

- 🎯 **完整电影工业化流程**：剧本 → 角色 → 场景 → 分镜 → Prompt → 生成
- 🤖 **7个专业AI Agent**：统筹 / 导演风格 / 剧本 / 角色设计 / 场景道具 / 分镜 / Prompt工程
- 🎨 **31位世界级导演风格库**：自动匹配最适合的导演组合
- 🔄 **3种协作模式**：顺序链 / 辩论投票 / 主席团（MoA）
- 📋 **结构化输出**：角色圣经 / 场景圣经 / 分镜表 / AI生成指令表
- 🌍 **多类型适配**：科幻、悬疑、爱情、动作、文艺、动画……
- 🎬 **双工具链支持**：GPT-image-2（图像生成）+ Seedance 2.5（视频生成）

## 🎭 Agent团队

### 🎯 流程统筹Agent (Executive Producer)
> "我是这部片子的执行制片人，所有部门的协调和质量由我把关"

- 解析用户需求 → 判断类型/风格/基调
- 组建虚拟剧组 → 匹配导演 + 各工种Agent
- 建立全局风格手册 → 所有Agent共享的风格基线
- 质量审查 → 确保角色/场景/风格一致性

### 🎨 导演风格Agent (Director Style)
> "我来帮你找到最适合这个故事的导演组合"

- 从31位导演风格库中匹配3-5位
- 输出视觉风格指南 + 叙事建议
- 输出色调/光影/构图方案
- 处理风格冲突与融合

**31位导演覆盖**：张艺谋、王家卫、诺兰、昆汀、黑泽明、宫崎骏、奉俊昊……

### 📖 剧本Agent (Screenplay)
> "把你的想法变成一个完整的故事"

- 三幕/五幕结构设计
- 场景分解（Scene Breakdown）
- 人物弧光设计
- 对白风格 + 节奏控制

### 👤 角色设计Agent (Character Design)
> "让你的角色从文字中走出来"

- 角色外观设计（年龄/体型/面容/发型）
- 服装风格 + 时代适配
- 角色圣经（Character Bible）
- GPT-image-2角色参考图prompt

### 🏗️ 场景道具Agent (Scene & Prop)
> "构建每个场景的世界，从大环境到小道具"

- 场景氛围设计（时间/地点/光线/天气）
- 色彩方案 + 空间布局
- 关键道具设计
- GPT-image-2场景/道具参考图prompt

### 🎬 分镜故事板Agent (Storyboard)
> "把剧本变成可拍摄的分镜脚本"

- 镜头选择（远景/中景/特写/大特写）
- 镜头运动（推/拉/摇/移/跟/升降）
- 构图方案（三分法/对称/引导线）
- 转场设计 + 时长估算
- Seedance视频生成prompt

### 🔧 Prompt工程Agent (Prompt Engineer)
> "把所有人的描述翻译成AI听得懂的语言"

- GPT-image-2优化prompt
- Seedance 2.5优化prompt（图生视频 + 文生视频）
- Negative prompts管理
- 跨场景一致性检查
- 参数建议（分辨率/帧率/seed等）

## 🤖 协作模式

### 模式A：顺序链（Sequential Chain）
> 按工种流水线：剧本 → 角色 → 场景 → 分镜 → Prompt

适合：明确的线性创作，步步为营

### 模式B：辩论投票（Debate & Voting）
> 3-5位导演各出方案，辩论后投票

适合：创意方案比选，集思广益

### 模式C：主席团（MoA - Mixture of Agents）
> 指定首席导演，各专家提供建议，首席整合

适合：复杂项目，需要统一美学风格

## 🔧 技术说明

### AI工具支持

| 工具 | 用途 | 模式 |
|------|------|------|
| **GPT-image-2** | 角色设计图 / 场景概念图 / 道具参考图 / 关键帧 | 文生图 |
| **Seedance 2.0/2.5** | AI视频片段生成 | 图生视频 + 文生视频 |

### Prompt语言策略
- Agent内部逻辑和描述：中文
- 输出给AI工具的prompt：**英文**（在GPT-image-2和Seedance上效果更好）
- 电影专业术语：保留英文（如 "dolly zoom", "rack focus", "Dutch angle"）
