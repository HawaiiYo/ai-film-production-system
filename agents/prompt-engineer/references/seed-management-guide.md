# Seed 值管理指南

> AI 视频/图像生成中 Seed 值的使用策略：维持一致性、可追溯性、可重复性。

## 一、Seed 基础概念

```
Seed（种子值）是 AI 生成模型的随机数初始值。
- 相同 prompt + 相同 seed = 几乎相同的输出
- 相同 prompt + 不同 seed = 不同变体
- 用于：一致性控制、批量生成、问题追溯
```

## 二、策略 A：Base Seed + 场景偏移（推荐）

```
Base Seed = [随机生成一个大整数，如：285639471]

分配规则：
├── 场景1所有镜头：Base Seed + 1~99
├── 场景2所有镜头：Base Seed + 100~199
├── 场景3所有镜头：Base Seed + 200~299
├── ...
├── 场景N所有镜头：Base Seed + (N-1)*100 ~ N*100-1
└── 独立道具/角色设计图：Base Seed + 10000+

同一场景内的分配：
├── 建立镜头（Establishing Shot）：场景Base Seed（如 Base+100）
├── 关键镜头（Key Shot）：场景Base Seed + 1~9
├── 过渡镜头（Transition）：场景Base Seed + 10~19
├── 特写镜头（Close-up）：场景Base Seed + 20~29
└── 其他镜头：场景Base Seed + 30~99
```

**优势**：
- 简单直观，便于追溯
- 同场景镜头 seed 相近，有助于视觉一致性
- 局部调整不影响其他场景

## 三、策略 B：按角色分配 Seed（角色一致性优先）

```
Base Seed = [随机生成]

角色 Seed：
├── 角色A：Base Seed + 1000
├── 角色B：Base Seed + 2000
├── 角色C：Base Seed + 3000
└── ...

角色A的所有出场镜头使用 Seed 范围：Base Seed + 1000~1099
→ 同角色不同场景的 Seed 相近，有助于维持角色一致性
```

**优势**：
- 角色一致性最强
- 适合以角色为主叙事的项目

## 五、Seed 与一致性关系

```
⚠️ Seed 不是万能的：

相同 Seed + 相同 prompt → 高度相似但不完全相同
├── 图像生成：~95% 相似
├── 视频生成：~70-85% 相似（取决于运动复杂度）
└── 角色面部：~60-80% 相似（最难的）

因此对于人脸的稳定性，优先使用：
1. 一致的固定描述片段（Fixed Description Fragment）
2. 相同的参考图（I2V 模式）
3. 相近的 Seed 值
4. 后期人工修复（极端情况下）

三要素结合（描述+参考图+seed）达到最佳稳定性。
```

## 六、Seed 使用决策流程

```
生成 AI 内容前：
├── 这是首次生成？
│   ├── YES → 使用 Base Seed + [场景/角色] 偏移
│   └── NO → 继续
├── 是迭代优化（同一prompt重试）？
│   ├── YES → 保持当前 Seed（可重现当前版本）
│   └── 可以微调 Seed 探索变体
├── 输出质量不满意？
│   ├── 描述问题 → 修改 prompt，保留 Seed
│   └── 随机性问题 → 微调 Seed（±10），保留 prompt
└── 追溯历史问题？
    └── 每个输出记录 prompt + Seed 组合
```

## 七、跨工具 Seed 管理

```
GPT-image-2 与 Seedance 的 Seed 通常独立：
├── GPT-image-2 的 Seed：控制图像初始随机状态
├── Seedance 的 Seed：控制视频初始随机状态
├── 同一镜头的 GPT-image-2 关键帧 Seed 与 Seedance I2V 的 Seed 可以不同
└── 但建议使用相同或相近 Seed 形成"锁定"关系

锁定关系表（可选）：
| 镜头 | GPT-image-2 Seed | Seedance I2V Seed |
|------|------------------|--------------------|
| S1-001 | 285639401 | 285639501 |
| S1-002 | 285639402 | 285639502 |
```

## 八、Seed 策略选择建议

| 项目类型 | 推荐策略 |
|---------|---------|
| 多场景多角色 | 策略 A（场景偏移） |
| 单角色多场景 | 策略 B（角色分配） |
| 标准商业项目 | 策略 A |
| 实验艺术项目 | 随机 Seed + 多次迭代 |
| 系列 IP（一二季） | 每季独立 Base Seed |
