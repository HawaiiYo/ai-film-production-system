# 箭头符号系统 (Arrow Notation System)

> 镜头运动 + 角色走位 + 复合标注的统一符号语言，供 storyboard Agent 与后续 AI 视频生成使用。

## 一、镜头运动箭头（Camera Movement Arrows）

镜头运动箭头画在**画框内部或边缘**，表示摄影机如何移动：

```
┌─────────────────────────────────────────────────┐
│              镜头运动箭头速查表                    │
├──────────┬──────────┬─────────────┬──────────────┤
│  运动     │  箭头符号  │  画法         │  含义          │
├──────────┼──────────┼─────────────┼──────────────┤
│ 推(Dolly │ →→→→→→  │ 向画面中心    │ 摄影机向前推进  │
│  In)     │ 汇聚箭头  │ 的四方向箭头  │                │
├──────────┼──────────┼─────────────┼──────────────┤
│ 拉(Dolly │ ←←←←←←  │ 从画面中心    │ 摄影机向后拉远  │
│  Out)    │ 发散箭头  │ 向外四散箭头  │                │
├──────────┼──────────┼─────────────┼──────────────┤
│ 左摇     │ ←─────── │ 画框顶部      │ 摄影机水平左转  │
│ (Pan L)  │ 左向箭头  │ 长箭头       │                │
├──────────┼──────────┼─────────────┼──────────────┤
│ 右摇     │ ───────→ │ 画框顶部      │ 摄影机水平右转  │
│ (Pan R)  │ 右向箭头  │ 长箭头       │                │
├──────────┼──────────┼─────────────┼──────────────┤
│ 上摇     │    ↑     │ 画框左侧      │ 摄影机仰起      │
│ (Tilt Up)│ 上向箭头  │ 垂直箭头     │                │
├──────────┼──────────┼─────────────┼──────────────┤
│ 下摇     │    ↓     │ 画框左侧      │ 摄影机俯下      │
│ (Tilt Dn)│ 下向箭头  │ 垂直箭头     │                │
├──────────┼──────────┼─────────────┼──────────────┤
│ 跟/轨    │ →→→→→→  │ 沿主体移动    │ 摄影机跟随主体  │
│ (Track)  │ 平行箭头  │ 方向的长箭头  │ 横向移动       │
├──────────┼──────────┼─────────────┼──────────────┤
│ 升降     │ ↑(Crane) │ 画框一侧      │ 摄影机垂直升降  │
│ (Crane)  │ ↓(Crane) │ 垂直箭头+标注 │                │
├──────────┼──────────┼─────────────┼──────────────┤
│ 手持     │ ≈≈≈≈≈≈  │ 画框边缘      │ 手持摄影晃动    │
│(Handheld)│ 波浪线    │ 波浪线       │                │
├──────────┼──────────┼─────────────┼──────────────┤
│ 变焦推   │ [Zoom    │ 画框四角      │ 焦距变长        │
│ (Zoom In)│  In]→    │ 向内箭头+标注 │                │
├──────────┼──────────┼─────────────┼──────────────┤
│ 变焦拉   │ ←[Zoom   │ 画框四角      │ 焦距变短        │
│(Zoom Out)│  Out]    │ 向外箭头+标注 │                │
└──────────┴──────────┴─────────────┴──────────────┘
```

**关键规则**：
- 箭头方向 = 摄影机移动方向（不是画面内容移动方向）
- 箭头长度 ≈ 移动的幅度（长箭头=大范围移动）
- 箭头粗细 ≈ 移动的速度（粗箭头=快速移动）
- 多箭头 = 复合运动（如推+摇用两个箭头组合）

## 二、角色/物体运动箭头（Character & Object Movement Arrows）

角色运动箭头画在**画面内部**，表示角色或物体在镜头内的运动轨迹：

```
┌──────────────────────────────────────────────────┐
│            角色/物体运动箭头速查表                  │
├────────────┬──────────┬──────────────────────────┤
│  运动类型   │  箭头符号  │  画法说明                  │
├────────────┼──────────┼──────────────────────────┤
│ 直线行走    │ ───→     │ 从角色出发的实线箭头       │
│ (Walk)     │          │ 箭头指向行走方向           │
├────────────┼──────────┼──────────────────────────┤
│ 奔跑       │ ═══▶     │ 粗实线箭头                │
│ (Run)      │          │ 比行走箭头粗2倍            │
├────────────┼──────────┼──────────────────────────┤
│ 曲线运动    │ ╭───╮   │ 虚线或细实线曲线箭头       │
│ (Curve)    │ ╯   ╰→  │ 表示绕行/迂回              │
├────────────┼──────────┼──────────────────────────┤
│ 进出画面    │ ←──●──→ │ 圆点=起点/终点             │
│ (Enter/Exit)│         │ 箭头指向移动方向           │
├────────────┼──────────┼──────────────────────────┤
│ 上下楼梯    │ ╱╱╱↑    │ 斜线+方向箭头              │
│ (Stairs)   │         │                           │
├────────────┼──────────┼──────────────────────────┤
│ 坠落/下落   │    ↓    │ 垂直向下粗箭头             │
│ (Fall)     │    ▼    │ 可加速度线                │
├────────────┼──────────┼──────────────────────────┤
│ 物体飞行    │ ~~~~→   │ 虚线+箭头                 │
│ (Projectile)│         │ 表示子弹/箭矢/投掷物       │
├────────────┼──────────┼──────────────────────────┤
│ 眼神/视线   │ ······→ │ 点线箭头                  │
│ (Eyeline)  │         │ 表示角色在看什么           │
└────────────┴──────────┴──────────────────────────┘
```

## 三、复合运动标注（Combined Movement Notation）

当一个镜头中同时存在多种运动时：

```
┌────────────────────────────────────┐
│  🎬 S1-005 (FS) — 5s              │
│                                    │
│       [场景线稿画面]                 │
│                                    │
│  📷 ←←← Dolly Out (摄影机后拉)     │
│  🚶 ───→ 主角向前走                │
│  👀 ····→ 主角看向右方             │
│                                    │
│  效果：Dolly Zoom效果               │
│  (摄影机后拉+变焦推进)              │
│  人物大小不变，背景压缩             │
└────────────────────────────────────┘
```

## Seedance Prompt 中的箭头翻译

将箭头符号转换为 Seedance prompt 中的镜头运动关键词：

| 箭头 | Seedance 关键词 |
|------|----------------|
| →→→→ Dolly In | `slow dolly in, [speed] movement, [distance]` |
| ←←←← Dolly Out | `slow dolly out, revealing [subject]` |
| ←── Pan L | `slow pan from right to left, [speed]` |
| ──→ Pan R | `slow pan from left to right, [speed]` |
| ↑ Tilt Up | `tilt up, revealing [above]` |
| ↓ Tilt Down | `tilt down, revealing [below]` |
| ▬▬→ Track | `tracking shot, following [subject], lateral movement` |
| ≈≈≈ Handheld | `handheld camera, subtle shake, documentary feel` |
| ↑↓ Crane | `crane shot, rising/falling, dramatic [position]` |
| [Zoom In] | `zoom in, focal length increasing, compressing space` |

**使用规则**：每个分镜表中的箭头必须能在 Seedance prompt 中找到对应的英文运动描述。
