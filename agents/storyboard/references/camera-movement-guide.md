# 镜头运动指南 (Camera Movement Guide)

> 镜头运动赋予画面生命力。不同的运动方式携带不同的情感信息。

## 镜头运动类型

| 运动方式 | 英文 | 技术描述 | 情感/心理效果 | 适用场景 |
|---------|------|---------|--------------|---------|
| 固定 | Static / Locked off | 摄影机完全不动，三脚架锁定 | 客观、冷静、观察者视角、庄严感 | 对话场景、需要观众专注内容的时刻、结尾定镜 |
| 推 | Dolly In / Push In | 摄影机沿轨道向主体推进 | 注意力集中、紧张感增加、进入角色内心 | 角色产生重要认知、情绪升级、揭示关键信息 |
| 拉 | Dolly Out / Pull Back | 摄影机远离主体，揭示更大环境 | 揭示、孤独感、抽离、宏大感 | 揭示角色所处环境的全貌、强调角色的渺小孤独 |
| 摇 | Pan Left/Right | 摄影机原地水平旋转 | 扫描、跟随视线、建立空间关系、过渡 | 跟随角色行走的视线、展示空间从左到右、反应镜头衔接 |
| 俯仰 | Tilt Up/Down | 摄影机原地垂直旋转 | 仰拍=崇敬/威胁/权力，俯拍=脆弱/渺小/全知 | 人物登场（仰）、揭示高度/规模（上下摇）、强调权力关系 |
| 跟 | Tracking / Follow | 摄影机跟随主体移动 | 代入感、流动感、与角色同行 | 角色行走/奔跑、探索新空间、动作追逐 |
| 升降 | Crane Up/Down | 摄影机在摇臂上垂直移动 | 史诗感、宏大的揭示或告别、上帝视角 | 场景建立、重要人物的退场、从微观到宏观的过渡 |
| 手持 | Handheld / Shaky Cam | 手持摄影，轻微抖动 | 真实感、紧张、纪录片质感、混乱 | 战争/打斗场面、主观视角、营造不安定感 |
| 变焦 | Zoom In/Out | 改变镜头焦距而非摄影机位置 | 突然的强调或揭示、突兀感（慎用） | 惊恐发现、喜剧效果、快速信息强调 |
| 环绕 | Orbit / Circle | 摄影机围绕主体360度旋转 | 包围感、全知视角、重要时刻的仪式感 | 关键对话高潮、角色觉醒时刻、重要决定的瞬间 |

## 运动设计原则

- 一个镜头的运动方式不超过1种主体运动（可以微弱的辅助运动）
- 运动速度与情绪匹配：紧张→快速推/跟，悲伤→缓慢拉/摇
- Seedance对推拉摇移跟表现良好，手持效果有波动
- 避免在同一镜头内做反转运动（先推后拉、先左摇后右摇）
- 静态镜头在Seedance中稳定性最好，动态镜头需要更详细的运动描述

## 情绪→运动映射

| 情绪 | 推荐镜头运动 |
|------|------------|
| 紧张/焦虑 | 缓慢推进(Dolly In)、手持微晃 |
| 悲伤/孤独 | 缓慢拉远(Dolly Out)、静态长镜头 |
| 喜悦/解放 | 上升(Crane Up)、快速跟拍 |
| 愤怒/冲突 | 快速推、手持震动、Dutch Angle |
| 平静/沉思 | 静态、缓慢横摇(Pan) |
| 恐惧/不安 | 手持、快速变焦、Dutch Angle |
| 浪漫/温馨 | 浅景深、缓慢环绕、柔光推镜 |
| 史诗/震撼 | 大景别升降、缓慢大范围摇移 |
