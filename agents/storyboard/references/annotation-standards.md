# 标注规范与速查 (Annotation Standards)

> 分镜编号体系、缩写速查表、标注规范。供 storyboard 内部使用。

## 一、分镜编号体系

### 镜头编号格式

```
大格式：S[场景]-[镜头序号]
├── S1-001       → 场景1，第1个镜头
├── S1-002       → 场景1，第2个镜头
├── S2-001       → 场景2，第1个镜头
└── S10-015      → 场景10，第15个镜头

小格式（精简）：可选用 [场景号].[序号]
├── 1.1          → S1-001
└── 10.15        → S10-015
```

### 场景编号

```
SCENE-1, SCENE-2, ... 或 S1, S2, S3, ...
├── SCN-1 = 场景 1，主场
├── SCN-2 = 场景 2
└── 可加扩展：SCN-1A, SCN-1B (同场景不同时段/角度)
```

### 镜头类型编号（可选）

```
在镜头序号后加类型后缀：
├── S1-001-E    → Establishing Shot 建立镜头
├── S1-002-K    → Key Shot 关键动作镜头
├── S1-003-C    → Close-up 特写
├── S1-004-T    → Transition 转场
└── S1-005-R    → Reaction Shot 反应镜头
```

## 四、时间标注速查

```
时分秒：
├── 小时:分钟:秒.帧
│   例：01:23:45.20  = 1小时23分45秒20帧
├── 总秒数
│   例：5247s = 87分27秒
└── Shot duration
    例：5s shot, 24fps = 120 帧
```

## 五、分镜信息标注规范（完整版）

```
每个镜头标注必须包含：

┌──────────────────────────────────────┐
│  镜头ID: S[场景]-[序号]                │
│  景别: [景别缩写+焦段]                │
│  运镜: [运镜描述]                    │
│  时长: [N]s                          │
│  帧率: [24/30/60]fps                │
├──────────────────────────────────────┤
│  构图: [构图类型+视点和角度]           │
│  光线: [光源方向+色温+强度]            │
│  色彩: [主导色彩+HEX]                │
├──────────────────────────────────────┤
│  角色: [出场的角色名]                  │
│  动作: [角色动作描述]                  │
│  对白(可选):                          │
│    - 中文原文                          │
│    - English (for AI)                │
├──────────────────────────────────────┤
│  音效: [环境音+音效+音乐提示]          │
│  道具: [画面中可见的关键道具]          │
│  备注: [额外说明]                     │
├──────────────────────────────────────┤
│  Seedance参数:                        │
│  - 模式: [T2V/I2V]                   │
│  - 参考图: [I2V时必填]                │
│  - Duration: [N]s                     │
│  - Motion Strength: [0-10]            │
│  - Seed: [数值]                       │
│  - Style Anchor: [全局风格锚点]        │
├──────────────────────────────────────┤
│  连贯性元数据:                         │
│  - 空间坐标: (x, y, z) 角色位置        │
│  - 视线方向: 左/右/前/后/上/下         │
│  - 光照方向: 与前镜一致性检查         │
│  - 时间: 与前镜头时间连续性            │
└──────────────────────────────────────┘
```

## 七、跨镜头标注一致性

```
确保以下标记在整部影片中保持一致：
├── 景别缩写 (WS / MS / CU ...)
├── 运镜缩写 (DI / DO / PAN ...)
├── 镜头编号 (S1-001, S2-005 ...)
├── 色彩 HEX 代码
├── 标准时长单位 (s)
└── Seed管理 (BASE + offset)
```
