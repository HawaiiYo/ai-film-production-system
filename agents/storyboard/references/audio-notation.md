# 声音标注系统 (Audio Notation)

> 分镜表中的声音元素标注规范，包括音乐、对白、音效、环境音。

## 一、声音元素分类

| 类型 | 简写 | 用途 |
|------|------|------|
| 对白 Dialogue | DLG | 角色说话 + 台词内容 |
| 旁白 Voice-over | V/O | 叙述者的旁白 |
| 音乐 Music | MUS | 背景音乐/主题音乐 |
| 音效 Sound Effect | SFX | 具体音效（爆破声、脚步声等） |
| 环境音 Ambient | AMB | 背景环境音（风、雨、虫鸣、城市） |
| 静默 Silence | SIL | 完全无声，刻意留白 |
| 配乐 Score | SCR | 原创配乐/管弦乐 |

## 二、分镜声音标注符号系统

```
┌──────────────────────────────────┐
│  符号     含义                       │
├──────────────────────────────────┤
│  )))     对白气泡 (连续音波)         │
│  )))N   旁白气泡                      │
│  ♪        背景音乐                   │
│  ♬↑      音乐渐强                    │
│  ♬↓      音乐渐弱                    │
│  *        音效                        │
│  ~        环境音/氛围音               │
│  ✕        完全静默                    │
│  !        突如其来的声音               │
└──────────────────────────────────┘
```

## 三、分镜表中声音标注格式

每个镜头的声音标注格式：

```
🎬 S1-005 (MS) - 5s
─────────────────────────
[画面描述]
─────────────────────────
🎵 ♪ 低沉大提琴主题
🌃 ~ 城市夜色背景声
💬 ))) "你确定吗?" (A低声)
🔊 * 远处轮胎摩擦声
─────────────────────────
```

## 四、Seedance Prompt 中的声音处理

```
⚠️ Seedance限制：
- Seedance不直接生成声音
- 声音在后期阶段添加

但Seedance prompt中应提示"预期声音"：
✅ "slow footsteps on gravel, distant city noise"
✅ "wind whistling through trees, rustling leaves"
✅ "tense silence, occasional drip of water"

作用：指导AI生成与声音匹配的画面（如：安静场景不会有突然强动作）
```

## 五、声音情绪映射表

| 情绪 | 推荐声音组合 |
|------|-------------|
| 紧张 | 低频嗡嗡声 + 偶尔心跳声 + 静默 |
| 悲伤 | 缓慢弦乐 + 雨声 + 静默 |
| 恐惧 | 极低频 + 突然尖锐声 + 静默 |
| 浪漫 | 柔和钢琴 + 环境白噪声 + 喘息 |
| 神秘 | 缓慢东方乐器 + 远处钟声 + 风 |
| 欢乐 | 节奏明快 + 笑声 + 热闹环境音 |

## 六、对白和台词在 Seedance Prompt 中

```
Seedance prompt 中不直接生成对白，但可以标注：
✅ "mouth moving as if speaking"
✅ "lips part as if talking"
✅ "without speaking, expression shows internal dialogue"

→ 通过视觉提示演员口型动作,避免不自然的对白生成
```
