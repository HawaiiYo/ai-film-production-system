# 转场设计 (Transition Types)

> 6 种核心转场类型及其节奏控制。供 storyboard Agent 在分镜表中标注转场使用。

## 一、基础转场类型速查

| 转场类型 | 英文名 | 视觉特征 | 节奏效果 | 适用情境 |
|---------|--------|---------|---------|---------|
| 硬切 | Hard Cut | 直接切换,黑场一闪即过 | 紧凑、利落 | 动作场面、节奏短拍 |
| 淡入/淡出 | Fade In/Out | 画面缓慢变暗/变亮 | 缓慢、庄重 | 时间跳跃、章节结束 |
| 叠化 | Dissolve | 前画面渐隐后画面渐显 | 柔和、梦幻 | 回忆、梦境、平行叙事 |
| 遮罩转场 | Wipe | 通过图形边界划过切换 | 节奏感强 | 风格化作品、MV |
| 划接 | Match Cut | 视觉元素相似性匹配 | 精巧、主题化 | 蒙太奇、隐喻 |
| 跳切 | Jump Cut | 同场景内时间跳跃 | 紧张、压抑 | 焦虑、时间紧迫 |

## 二、转场用途决策表

| 叙事需求 | 推荐转场 |
|---------|---------|
| 时间跨度大（几年/几天） | 淡入淡出 / 章节标题 |
| 时间跨度中等（几小时） | 硬切 + 时间点字幕 |
| 时间跨度小（几十秒） | 叠化/匹配剪辑 |
| 情绪突变/揭示 | 硬切 |
| 回忆/梦境 | 叠化 + 色调偏移 |
| 并列叙事/对照 | 跳切 |
| 空间跳跃 | 硬切 |

## 三、Seedance 转场约束

```
⚠️ Seedance转场限制：
- 单个镜头不包含转场（Seedance生成单个连续片段）
- 转场需要在后期制作中合成（不在分镜表中执行）
- Seedance只负责"镜头内的状态变化"（如角色进入状态A → 状态B）
- "镜头间转场"交给后期编辑阶段
```

## 四、镜头内的"伪转场"技巧

在 Seedance 单镜头内模拟转场感：

| 技巧 | 描述 | Seedance 关键词 |
|------|------|----------------|
| 时空跳跃 | 镜头内从白天告诉变到黑夜 | `time passes, daylight fades to night, gradual transition within shot` |
| 黑闪 | 突然黑场1秒后回到画面 | `brief blackout, flash cut, sudden dark, return to scene` |
| 抽帧 | 几个画面快速闪过 | `quick montage flashes, brief snapshots` |
| 渐变调色 | 镜头内色调从暖变冷 | `color temperature shift, gradually cooling tones` |

## 五、分镜表中转场标注

```
转场标记：
├── CUT         → 硬切 (最常用)
├── FADE        → 淡入淡出
├── DISS        → 叠化  
├── MATCH       → 匹配剪辑
├── WIPE        → 遮罩转场
├── JUMP        → 跳切
└── INTERNAL    → 镜头内转场 (Seedance受限)
```
