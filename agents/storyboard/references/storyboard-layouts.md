# 分镜布局系统 (Storyboard Layout Types)

> 6 种分镜板式布局 + 选择指南。供 storyboard Agent 输出分镜表时选择使用。

## 一、分镜布局类型选择

| 布局类型 | 镜头数/板 | 信息深度 | 适用场景 |
|---------|---------|---------|---------|
| 单格布局 (Single Panel) | 1 | 极深 | 关键镜头、节奏重点 |
| 双格对称布局 (Split) | 2 | 深 | 平行叙事、对比 |
| 三格连续布局 (Strip) | 3-4 | 中 | 动作序列、追逐 |
| 四宫格布局 (Grid 2×2) | 4 | 中 | 同场景多视角 |
| 五格电影感布局 (Cinematic) | 5 | 浅-中 | 主要场景 |
| 六格密集布局 (Dense) | 6 | 浅 | 超高信息密度场景 |

## 二、布局 1：单格布局 (Single Panel)

```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│                                     │
│         [LARGE SKETCH]              │
│                                     │
│                                     │
│                                     │
│  ACTION: ________________________  │
│  CAMERA: ________________________  │
│  DIALOGUE: _____________________   │
└─────────────────────────────────────┘
```

**适用**：关键揭示镜头、情感最高点、需要特别关注的镜头

## 三、布局 2：双格对称布局 (Split Panel)

```
┌─────────────────────────────────────┐
│                  │                  │
│                  │                  │
│   [LEFT FRAME]   │   [RIGHT FRAME]  │
│                  │                  │
│                  │                  │
│  ACTION: ______  │  ACTION: ______  │
│  CAMERA: ______  │  CAMERA: ______  │
└─────────────────────────────────────┘
```

**适用**：打电话对话、平行叙事、双人视角对照

## 四、布局 3：三格连续布局 (Strip)

```
┌─────────────────────────────────────┐
│                  │                  │
│ [FRAME 1]        │ [FRAME 2]        │ [FRAME 3]
│                  │                  │
├──────────────────┼──────────────────┤
│  ACTION: A→B     │ B→C              │ C action │
└─────────────────────────────────────┘
```

**适用**：连续动作、运动镜头、追逐

## 五、布局 4：四宫格布局 (Grid 2×2)

```
┌──────────────────────────────────────┐
│                  │                   │
│   [FRAME 1]      │   [FRAME 2]       │
│   镜头1: 远景     │   镜头2: 中景      │
│                  │                   │
├──────────────────┼───────────────────┤
│                  │                   │
│   [FRAME 3]      │   [FRAME 4]       │
│   镜头3: 近景     │   镜头4: 特写      │
│                  │                   │
└──────────────────────────────────────┘
```

**适用**：同场景多视角、标准叙事段落

## 六、布局 5：五格电影感布局 (Cinematic)

```
┌──────────────────────────────────────┐
│                  │                   │
│   [镜头1 大格]    │   [镜头2]          │
│                  ├───────────────────┤
│   建立镜头/CAM1    │   [镜头3]          │
│                  │                   │
├──────────────────┴───────────────────┤
│  [镜头4]   │ [镜头5]                 │
│           │                          │
└──────────────────────────────────────┘
```

**适用**：完整场景单元、镜头组合多样 + 主镜头突出

## 七、布局 6：六格密集布局 (Dense)

```
┌──────────────────────────────────────┐
│   [1]  │ [2]  │ [3]                  │
│        │      │                       │
├────────┼──────┼───────────────────────┤
│   [4]  │ [5]  │ [6]                  │
│        │      │                       │
├────────┴──────┴───────────────────────┤
│  镜头编号 / 描述 / 音效 / 备注           │
└──────────────────────────────────────┘
```

**适用**：高密度分镜、动作场面、复杂蒙太奇

## 八、布局选择决策树

```
选择分镜布局：
├── 这镜头是关键镜头/重要揭示？
│   └── YES → 单格布局 
├── 平行叙事/双视角？
│   └── YES → 双格布局
├── 动作序列/追逐？
│   └── YES → 三格布局
├── 标准多镜头场景？
│   └── YES → 四宫格布局
├── 复杂场景/多视角？
│   └── YES → 五格布局
├── 超高密度/动作场面？
│   └── YES → 六格布局
└── 默认 → 四宫格布局
```

## 九、布局输出格式规范

每个分镜布局的输出须包含：
- **视觉区**：画面占 60-70% 空间
- **标注区**：动作/对话/镜头 描述占 30-40%
- **统一格式**：
  ```
  镜头ID: S[场景]-[序号]
  景别: [ELS/LS/MS/CU/...]
  动作: [中文动作描述]
  对话: [中文对话 + 英文台词]
  镜头: [运镜描述]
  音效: [环境声/音乐提示]
  时长: [N]s
  ```

---

## 输出形式

分镜表的最终输出可以是：
- **Markdown 表格**（最常用，便于阅读和编辑）
- **纯文本卡片**（每镜头单独块，便于提取）
- **可视化 ASCII 布局**（增强可读性）
