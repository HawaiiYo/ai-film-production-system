## 二、常见场景材质库

| 材质类型 | 子类型 | 视觉特征 | 常见用途 |
|---------|--------|---------|---------|
| 木材 | 原木/漆木/旧木/朽木 | 纹理走向、颜色深浅、光泽度、裂纹 | 地板/家具/梁柱 |
| 石材 | 大理石/花岗岩/石灰石/砖 | 纹理、结晶、颜色、风化程度 | 地面/墙面/台面 |
| 金属 | 不锈钢/黄铜/铁锈/铝合金 | 反射率、颜色、氧化程度、加工痕迹 | 结构/装饰/家具 |
| 玻璃 | 透明/磨砂/彩色/镜面 | 透明度、反射、折射、厚度 | 窗/门/隔断/装饰 |
| 织物 | 棉/麻/丝/绒/皮革 | 织法、光泽、垂坠感、磨损 | 窗帘/地毯/家具软包 |
| 墙面 | 涂料/壁纸/裸露砖/混凝土 | 纹理粗细、颜色、修补痕迹 | 墙面/天花板 |
| 地面 | 木地板/水泥/瓷砖/石材/地毯 | 拼接方式、磨损程度、反光 | 地面 |
| 自然 | 泥土/草地/沙地/岩石/水面 | 湿度、颗粒大小、植被覆盖 | 室外场景 |

## 三、材质老化/磨损程度分级

| 级别 | 描述 | 视觉特征 | 适用场景 |
|------|------|---------|---------|
| **全新** | 刚刚建成/购买 | 无磨损、光泽完整、颜色饱满 | 新房子、未来场景 |
| **轻度使用** | 正常使用痕迹 | 轻微磨损、少量划痕、边角微损 | 普通家庭、办公室 |
| **中度老化** | 明显使用多年 | 可见划痕、褪色、局部损坏 | 老房子、旧公寓 |
| **重度老化** | 年久失修 | 大面积损坏、严重褪色、结构不稳 | 废弃建筑、贫民窟 |
| **废墟级** | 完全废弃 | 结构坍塌、自然入侵、只剩框架 | 末日场景、历史遗迹 |

## 五、材质使用决策树

```
选择材质方向：
├── 时代背景？
│   ├── 古代 → 自然材质（木材/石材/布）+ 重度老化
│   ├── 近代 → 木材/石材 + 混合老化
│   ├── 现代 → 多材质 + 轻度老化
│   └── 未来 → 合成材料 + 全新/磨损同存
├── 场景情绪？
│   ├── 温暖 → 木/布/纸（哑光为主）
│   ├── 冷峻 → 金属/玻璃/混凝土
│   ├── 古旧 → 重度老化纹理
│   └── 未来 → 光滑合成材料
└── 角色定位（重要道具）？
    ├── 主角 → 中等老化 + 完整性高
    ├── 反派 → 冷调材质（金属/玻璃）
    └── 平民 → 自然材质 + 中度老化
```

## 六、材质与光线互动

```
不同材质对光线的反应：

高反射率（金属、玻璃）：
- 显示环境反射
- 高光点锐利
- 受光源方向影响大

中反射率（木材、石材墙）：
- 显示光照方向
- 中等高光强度

低反射率（织物、地毯）：
- 漫反射为主
- 几乎没有高光点
- 显示颜色而非光泽

透明/半透明（玻璃、宣纸）：
- 光线穿透
- 显示背后物体
- 有特殊光晕效果
```

## 七、材质与情绪映射

| 情绪 | 推荐材质组合 |
|------|-------------|
| 温暖/怀旧 | 旧木、亚麻布、皮革、纸质、铜锈 |
| 冷峻/未来 | 不锈钢、玻璃、合成材料、抛光金属 |
| 神秘/古老 | 风化石材、氧化金属、苔藓、古砖 |
| 紧张/紧迫 | 工业金属、混凝土、生锈铁、玻璃碎片 |
| 浪漫/亲密 | 丝绸、天鹅绒、暖色木材、蜡烛光下的布料 |
| 恐怖/阴暗 | 腐烂织物、剥落墙皮、发霉石材、生锈管道 |
