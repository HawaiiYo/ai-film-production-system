## 使用说明

```
1. 统筹 Agent 收到项目需求 → 分析流派/情绪/主题
2. director-style Agent 根据索引选择 3-5 位导演
3. 在 `./[导演名].md` 加载详细风格描述
4. 输出视觉风格指南和风格锚点（Style Anchor）
```

**注意事项**：
- 每个导演文件含六大模块：核心创作思想、美学特征、叙事手法、人物塑造、代表作品风格解析、多 Agent 启发
- 加载导演文件只需 Read 其内容，无需 Learn/Memorize，全部缓存到对话上下文
- 风格冲突时参考 director-style Agent SKILL.md 中的"风格冲突解决策略"
