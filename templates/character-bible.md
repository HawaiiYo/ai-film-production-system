# 👤 角色圣经模板 — Character Bible Template

> 为每个角色建立固定视觉描述,跨场景复用,确保AI生成的一致性
>
> **使用流程**:
> 1. 角色设计 Agent 沿本模板填写
> 2. 填到 `🌐 Prompt 生成偏好` 时**必须先询问用户**确认语言版本/情绪选择/格式等
> 3. 章节使用规范:除特别说明外,所有表格的"内容/描述"列必须填满;

---

### 🌐 Prompt 生成偏好 ⭐必填

> **本节必须在生成任何 Prompt 之前由输出方反问用户确认。**
> 不确认则不允许直接进入 `🌐 GPT-image-2 Prompt` 章节,只先输出档案部分。

| 属性 | 默认值 | 询问选项 |
|------|--------|---------|
| Prompt 语言版本 | 待用户确认 | 中文版 / 英文版 / 双语并列 |
| Prompt 格式 | 长 prompt 单条 | 单条长 prompt / 分件(多个) / 视情况 |
| 情绪变体选择 | 七种全选 | 全部七种 / 仅核心(悲伤等)/ 按剧本节点选 |
| Negative 提示词 | 中文 | 中文 / 英文 / 双语 |
| 风格锚点 | 由导演风格 Agent 提供 | 已锁定导演风格 / 由用户口述 |
| 衍生 Prompt 是否本角色 Agent 产出 | 否(只标注来源) | 仅引用 / 部分给出 / 全部生成 |
| 是否输出"主体物/配饰锚点特写" | 是 | 是 / 否 / 仅主体物 |
| 是否输出"三阶段演化图" | 是 | 是 / 否 |

> **确认机制**:输出方须用 `AskUserQuestion` 在生成 Prompt 前至少询问:
> (1)语言版本;(2)情绪选择范围;(3)是否需要本 Agent 即时生成衍生 Prompt。

### 🔗 主体物/配饰锚点 ⭐新增

> **全片反复出现的视觉物件清单**。在跨场景 prompt 中必须出现,用以建立"主体感"。
> 拍摄前由本档案定锚,**不允许下游 Agent 临时改动**。

| # | 主体物名称 | 实体描述 | 叙事意义 | 出现频率 | 是否每镜必出现 |
|---|------------|---------|---------|---------|--------------|
| 1 | [例:红色旧舞鞋] | [缎面/略旧/红色] | [梦想+妈妈的远方之手] | [全片出现 N+ 次] | □ 是 □ 否 |
| 2 | [配件 1] | | | | □ 是 □ 否 |
| 3 | [配件 2] | | | | □ 是 □ 否 |

> **使用规范**:
> - 每个主体物最好有独立的"道具锚点特写 Prompt",见下方 `🧩 衍生 Prompt 池`
> - "每镜必出现"的锚点视为**视觉锁**,分镜 Agent 引用本节作为强制约束

### 🖼️ 可视化布局 ⭐新增(ASCII 框图 + 复选框)

> 本节定义**该角色的视觉资产清单**。下游视觉环节(导演风格 Agent / 角色设计图环节)
> 按此布局产出对应图像。框图是"视图结构图",复选框是"出图开关"。

```
┌─ 🖼️ 可视化布局 ─────────────────────────────────────┐
│                                                      │
│  □ 三视图(正面 / 四分之三 / 侧面 / 背面)             │
│                                                      │
│  □ 7 表情合集(横排,光照统一)                         │
│                                                      │
│  □ 阶段演化图(2–4 阶段并排,中间细线分隔)            │
│                                                      │
│  □ 道具/主体物锚点特写(每物一张)                     │
│                                                      │
│  □ 标志性姿态参考(动作预算图 / 概念草图)             │
│                                                      │
│ ─────────────────────────────────────────────────  │
│  总开关:□ 全部勾选   □ 按预算选   □ 仅核心           │
└──────────────────────────────────────────────────────┘
```

> **使用规范**:
> - 勾选项目进入下方 `🌐 GPT-image-2 Prompt` 章节
> - 不勾选项目仅作档案参考
> - 在 `🌐 Prompt 生成偏好` 中应再次确认

### 🌐 GPT-image-2 Prompt

> 本节按 `🌐 Prompt 生成偏好` 一节的确认结果输出:
> - 偏好在"中文版"则只输出中文版
> - "英文版"则只输出英文版
> - "双语并列"则两者前后并列

#### 角色设计图 Prompt(角色卡 · 主体输出)

> 用于一次性生成"角色设计图 + 表情集 + 阶段演化图 + 主体物"组合。具体要全包含或部分包含,以可视化布局勾选项为准。

```
[中/英文版依据偏好]

本 Prompt 必须包含:
- 主体描述(年龄/性别/民族/身高/体型)
- 面部 8 要素全填(脸型/肤色/眼睛/眉毛/鼻子/嘴唇/发型/特殊标记)
- 完整服装(上衣/下装/外套/鞋子/配饰)
- 主体物/配饰锚点(见 🔗 节)
- 姿态 + 表情(可指定一组或多组,按情绪选择偏好)
- 光线方案(柔侧光/冷白光/舞台暖金光 等)
- 视觉风格参考(见 🎨 节)
- 版式规格(8K / 多视图 / 表情合集 等)
```

#### 角色定妆照 Prompt(主形象锁定)

> 用于锁定主定妆照,作为跨镜一致性参考。**这是必出项**。

```
[中/英文版依据偏好]

本 Prompt 必须包含:
- 主角中近景或半身景
- 面部重点重述(痣/雀斑等关键识别点必须出现)
- 上半身服装完整描述
- 表情设定(默认:常态;可选:沉思版本作为"角色本质照")
- 光线方案比设计图更戏剧化
- 构图方案(中近景/平视/焦段/景深)
```

> **格式规范**:
> - 中文版需使用"胶片感 / 八视图 / 漫射光 / 次表面散射 / 光线追踪"等标准术语
> - 英文版按 GPT-image-2 习惯格式书写
> - Negative 提示词按偏好(中文/英文/双语)跟在主 Prompt 后

### 🧩 衍生 Prompt 池(引用 + 来源标注) ⭐新增

> 本节**不写出完整 Prompt 文本**,只列出**会被独立产出的 Prompt 清单**。
> 每个衍生 Prompt 在外部(MD 文件、其他 Agent 产出)中**独立存在**,本卡只引用与标源。

| # | 衍生 Prompt | 触发条件 | 产出文件路径 / 所属 Agent | 优先等级 |
|---|------------|---------|--------------------------|---------|
| 1 | 阶段演化图 Prompt | 可视化布局 □ 勾选 | 角色 Agent 自产出,或:`prompt-engineer/references/` | P1 |
| 2 | 主体物/配饰锚点特写 Prompt | 可视化布局 □ 勾选 | 角色 Agent 自产出,或:`scene-prop/` | P1 |
| 3 | 标志性姿态参考 Prompt | 可视化布局 □ 勾选 | 角色 Agent 自产出,或:`storyboard/` | P2 |
| 4 | 镜头级 Prompt(拍摄分镜) | 分镜阶段触发 | `storyboard/SKILL.md` 主控 | P2 |
| 5 | 情绪变体 Prompt(7 表情逐一) | Prompt 偏好 □ 勾选 | 角色 Agent 自产出,或:`prompt-engineer/` | P3 |
| 6 | 跨语言版本 Prompt(中英之外) | 用户特殊要求时 | 暂未实现,留空 | - |

> **使用规范**:
> - 衍生 Prompt 只在 `🌐 Prompt 生成偏好` 选择"是/部分给出"时,由本 Agent 即时生成;
>   否则只生成引用,具体生成交由对应 Agent 触发
> - 每个衍生 Prompt 都必须**复用主体物/配饰锚点**作强约束
> - 衍生 Prompt 输出位置默认:`agents/character-design/outputs/{角色名}/`

### 📋 一致性检查清单

#### A. 基础一致性(原模板 6 项保留)

- [ ] 面部特征描述是否足够具体(能否让 AI 重现相似的面孔)?
- [ ] 服装描述是否覆盖所有主要场景?
- [ ] 表情描述是否覆盖剧本中的关键情绪节点?
- [ ] 体态描述是否体现角色性格?
- [ ] GPT-image-2 prompt 是否包含画质参数?
- [ ] 角色描述是否与导演风格指南一致?

#### B. 主体物锚点一致性 ⭐新增

- [ ] `🔗 主体物/配饰锚点` 中标为"每镜必出现"的项,是否在所有 prompt 中都出现?
- [ ] 主体物在不同场景的状态变化(如红舞鞋:崭新→磨损→收藏→重启→封存)是否与剧本时点一致?
- [ ] 每件主体物是否都有独立的衍生 Prompt 引用?

#### C. 跨阶段一致性 ⭐新增

- [ ] 面部特征(痣/雀斑/瞳色等)在所有阶段演化图中是否完全一致?
- [ ] 跨阶段"必出现锚点"(如苗银项圈)是否在所有演化阶段都保留?
- [ ] 弧光视觉映射中的服装/表情/体态变化是否有叙事支撑?

#### D. Prompt 偏好一致性 ⭐新增

- [ ] 是否在生成任何 Prompt 前用 AskUserQuestion 询问过偏好?
- [ ] 语言版本选择(中/英/双语)是否与下游消费方匹配?
- [ ] 是否避免了"未询问直接输出默认版本"?
