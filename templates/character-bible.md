## 🌐 GPT-image-2 Prompt

### 角色设计图Prompt（英文）

```
Character Design Sheet: [角色英文名]

[完整英文角色描述，包含：
- 基本信息和体型
- 面部特征详细描述
- 服装完整描述
- 发型描述
- 姿势和表情]

[光线方案描述]

Character design sheet, full body, front view + side view + back view,
multiple expression variants (neutral, happy, angry, sad).

8K resolution, hyperrealistic, professional character concept art,
in the style of [风格参考].
```

### 角色定妆照Prompt（英文）

```
Character Portrait: [角色英文名]

[面部重点描述]
[上半身服装描述]
[表情描述]

[光线方案], [构图].

Cinematic portrait, medium close-up, shallow depth of field,
8K resolution, hyperrealistic, in the style of [风格参考].
```

## 📋 一致性检查清单

- [ ] 面部特征描述是否足够具体（能否让AI重现相似的面孔）？
- [ ] 服装描述是否覆盖所有主要场景？
- [ ] 表情描述是否覆盖剧本中的关键情绪节点？
- [ ] 体态描述是否体现角色性格？
- [ ] GPT-image-2 prompt是否包含画质参数？
- [ ] 角色描述是否与导演风格指南一致？
