## 📝 执行建议

### 生成顺序建议

```
推荐生成顺序（最大化一致性）：

第一步：角色设计图
  → GPT-image-2生成所有角色的设计图
  → 确认角色视觉形象

第二步：场景概念图
  → GPT-image-2生成所有场景的概念图
  → 确认场景视觉方向

第三步：场景关键帧
  → GPT-image-2为每个场景生成关键帧
  → 这些关键帧将作为Seedance图生视频的参考

第四步：视频生成（按场景顺序）
  → 场景1的所有镜头 → 检查一致性 → 调整
  → 场景2的所有镜头 → 检查一致性 → 调整
  → ...

第五步：后期拼接
  → 将所有视频片段导入剪辑软件
  → 按分镜表拼接
  → 添加转场、调色、音效
```

### Seed值管理建议

| 内容 | Seed建议 |
|------|---------|
| 同一角色的不同设计图 | 相同seed |
| 同一场景的不同角度 | 相同seed |
| 不同场景 | 不同seed |
| 同一场景的连续镜头 | 相同或相近seed |

### 分辨率与画质建议

```
- 所有GPT-image-2输出：使用最高可用分辨率
- 所有Seedance输出：根据最终成片目标选择
  - 短视频平台：1080p, 30fps
  - 电影级：4K, 24fps
- 建议先生成1080p测试效果，确认后用4K生成最终版
```
