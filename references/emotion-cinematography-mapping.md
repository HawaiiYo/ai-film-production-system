# 🎭 情绪→摄影映射知识库

> Emotion-Cinematography Mapping Reference — 情绪如何转化为具体的视觉/听觉决策

---

## 一、情绪维度模型（VAD+Intensity）

每个情绪在以下4个维度上定位：

```
Valence（愉悦度）：  -5 ←─── 不悦 ─── 中性 ─── 愉悦 ───→ +5
Arousal（唤醒度）：  -5 ←─── 平静 ─── 中性 ─── 兴奋 ───→ +5
Dominance（支配度）：-5 ←─── 被控 ─── 中性 ─── 掌控 ───→ +5
Intensity（强度）：   0 ←─── 微弱 ─── 中等 ─── 强烈 ───→ 10
```

## 二、8大基础情绪 → 摄影决策映射

### 😡 愤怒（Anger）
```
VAD坐标：    Valence=-4  Arousal=+4  Dominance=+3  Intensity=8
时间感知：   加快（主观时间变慢，剪辑快）
空间感知：   狭窄（隧道视野）
```

| 摄影维度 | 建议 | 说明 |
|---------|------|------|
| **景别** | CU→ECU | 逼近面部，压迫感 |
| **镜头运动** | 快速推镜(Dolly In) / 手持晃动(Handheld) | 侵略性或失控感 |
| **构图** | Dutch Angle(斜角) / 对角线 / 中心压迫 | 失去平衡 |
| **焦段** | 24-35mm（靠近变形） | 夸大人物的压迫感 |
| **光线** | 硬光 / 侧光 / 底光 / 高对比度 | 强烈阴影，攻击性 |
| **色调** | 暖红偏色 / 去饱和红 / 暗部发红 | 血液/火焰联想 |
| **剪辑节奏** | 快切 / 跳切 / ASL 1-3s | 急促呼吸感 |
| **声音** | 低频轰鸣 / 打击乐 / 呼吸声增强 | 心跳加速 |
| **Seedance参数** | Motion:0.8-1.0 / Duration:2-4s | 高速运动 |

**导演参考**：朴赞郁的对称暴力、大卫·芬奇的压抑愤怒、昆汀的爆发式愤怒

## 三、情绪过渡映射

情绪不是静止的——场景中最重要的是**情绪的转变**。

### 3.1 情绪转换类型

| 转换类型 | 摄影处理 | 示例 |
|---------|---------|------|
| 平静→紧张 | 构图逐渐收缩+光线逐渐变暗+镜头逐渐逼近 | 发现异常 |
| 喜悦→悲伤 | 色调从暖转冷+镜头从广转长+景别从远转近 | 得知噩耗 |
| 恐惧→愤怒 | 镜头从被动POV→主动推进+光线从暗→硬+从被控→掌控 | 反击时刻 |
| 紧张→释放 | 镜头突然拉开+光线变亮+构图打开+长呼气式剪辑 | 危机解除 |
| 沉思→顿悟 | 固定→突然推近+浅景深→深焦+去饱和→恢复色彩 | 想通了 |
| 震惊→接受 | Crash Zoom→缓慢后退+高对比→柔化+中心→三分法 | 消化冲击 |

### 3.2 情绪过渡技巧

```
技巧1：匹配剪辑（Match Cut）
从愤怒的紧握拳头 → 切到 → 悲伤的松开手掌
（手的形状匹配，情绪通过手势转变呈现）

技巧2：叠化（Dissolve）
在情绪转变点使用叠化来柔化过渡
愤怒的人脸 叠化到 悲伤的同一张脸

技巧3：色调渐变
在连续镜头中微调色调参数：
镜头1: 冷蓝 80% → 镜头2: 冷蓝 60% → 镜头3: 暖黄 40% → ...
（让色调变化跟随情绪变化，观众潜意识感知）

技巧4：景别呼吸
紧张: CU → MCU → MS → LS（释放）
进入: LS → MS → MCU → CU（逼近）
```

## 四、复合情绪处理

真实场景很少是单一情绪。复合情绪的处理原则：

### 4.1 情绪优先级
```
主导情绪 > 次要情绪

规则：主导情绪决定基础摄影方案，次要情绪提供微妙调整
示例：悲伤+愤怒
  → 基础（悲伤）：长焦 / 柔光 / 去饱和
  → 调整（愤怒）：加10%手持晃动 / 加速5%剪辑 / 暗部微红
```

### 4.2 矛盾情绪（Ambivalence）
```
又爱又恨 / 喜极而泣 / 甜蜜的忧伤

处理：使用视觉矛盾
- 暖色调 + 荷兰角（甜蜜中的不安）
- 灿烂阳光 + 极浅景深（明亮中的孤立）
- 欢快音乐 + 缓慢剪辑（快乐下的隐痛）
```

## 六、情绪一致性验证规则

### 规则1：场景内情绪参数不跳跃
同一场景内，相邻镜头的情绪参数差异不应超过2个等级
❌ 镜头3：CU + 手持晃动（恐惧8级）→ 镜头4：ELS + 静态（平静2级）[无过渡]
✅ 镜头3：CU + 手持晃动（恐惧8级）→ 镜头4：MCU + 微晃手持（紧张5级）[过渡]

### 规则2：情绪转折需要缓冲镜头
情绪从一种状态到相反状态时，至少需要1个过渡镜头
❌ 直接切换：极度愤怒 → 平静微笑
✅ 缓冲：极度愤怒 → 茫然/疲惫 → 平静微笑

### 规则3：全局情绪曲线合理
全片的情绪曲线应有起伏但不过度跳跃
✅ 正常曲线：悲→紧→释→喜→紧→怒→悲→静
❌ 异常曲线：悲→怒→悲→怒→悲→怒（情绪振荡）
